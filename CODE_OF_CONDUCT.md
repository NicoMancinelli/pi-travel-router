# Code of Conduct

## Scope

This code of conduct applies to all project spaces: GitHub Issues, Pull Requests, Discussions, code comments, and any other channel where project participants interact.

## Expected behaviour

- Be respectful and constructive. Disagreement is fine; personal attacks are not.
- Give and receive feedback graciously. Assume good intent.
- Stay on topic. Keep discussions relevant to the project.
- Welcome newcomers and answer questions patiently.

## Unacceptable behaviour

- Harassment, intimidation, or personal attacks of any kind.
- Hate speech or discriminatory language based on identity, background, or beliefs.
- Sharing others' private information without consent.
- Sustained disruptive behaviour after being asked to stop.

## Enforcement

Report concerns to **nicomancinelli@gmail.com**. The maintainer will review reports and respond within 7 days. Reports are treated confidentially.

Depending on severity and context, violations may result in a warning, temporary suspension from participation, or a permanent ban from all project spaces. The maintainer has final discretion.

## Note

This is a small hobbyist project. The goal is a welcoming space for people who want to build cool things together. Most interactions here are friendly and low-key — this document just sets the floor.

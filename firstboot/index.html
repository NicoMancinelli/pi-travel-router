Travel Router Setup

First-time configuration. Fields marked * are required; everything else is optional.

Please fix the following:
    Wi-Fi access point 1-32 characters. The Wi-Fi network name your devices will connect to.
    8-63 characters. Used by clients joining your AP. Sets the Wi-Fi regulatory domain (allowed channels and power levels). Choose the country where the router is used. Used for scheduled AP disable timers and the daily digest. Default is UTC. Used for mDNS (hostname.local). Default: travelrouter. Letters, numbers, hyphens only.
    Advanced (optional)
    Push notifications for WAN/VPN events. Leave blank to disable.
    Pre-authorize the Pi onto your tailnet. Leave blank to authenticate manually later. If using self-hosted Headscale instead of Tailscale cloud.
    Paste your ed25519/rsa public key. Leave blank to keep password authentication.
    Image ships with root/changeme. Set a new password (8+ chars) to change it now, or leave blank. Strongly recommended — the default password is 'changeme' Must match the password above.
    Find in iOS Settings → General → About → Bluetooth. Format: AA:BB:CC:DD:EE:FF
    Extra arguments passed to tailscale up. Common additions: --exit-node=, --ssh.
    Optional. Pushes vnStat metrics to a Prometheus push gateway. Leave blank to disable. Battery percentage at which the Pi safely shuts down (PiSugar boards only). Default: 10%.
    Features
    Times are in router local time (set timezone above).
    e.g. 50mbit, 10mbit, unlimited
    Space-separated list. Required when split tunnel is enabled.
    Passphrase for the separate Tor AP SSID. Required when Tor proxy is enabled.
    Space-separated MAC addresses to route through Tailscale. Leave blank = all devices.
    Setup takes about 5-10 minutes. The Pi will reboot automatically when finished — reconnect to your new SSID afterward.
